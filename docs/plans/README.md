# 计划索引

本目录用于非平凡的实现计划。

- 从 `00-plan-authoring-and-execution-guide.md` 开始
- 每个计划只聚焦一个结果面
- 归档或替换计划时按状态处理，不要删除历史
- 对于小型和中型项目，优先使用 `docs/plans/YYYY-MM-DD-topic-plan.md`
- 如果计划数量变多，可以接受使用数字前缀，如 `NNN-YYYY-MM-DD-topic-plan.md`

任何创建的计划，在完成前都必须先通过计划审计，再通过收尾审计。

推荐文件名：

- `2026-05-21-user-list-plan.md`
- `2026-05-21-role-permission-alignment-plan.md`
- `012-2026-05-21-checkout-validation-plan.md`
